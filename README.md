# TESLO API
1. clonar proyecto.
2. ```yarn install```
3. clonar  el archivo ```env.template``` y renombrarlo a ```.env```
4. cambiar las variables de entorno
5. levantar docker
```
docker-compose up - d
```

6. ejecutar seed
```
localhost:3000/api/seed
```

7. levantar
```
yarn start:dev
```
